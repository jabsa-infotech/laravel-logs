# laravel-logs

A simple package to check production logs

## Installation

You can install the package via composer:

```bash
composer require jabsa/laravel-logs
```

## Usage

```php
use jabsa\LaravelLogs\Facades\LaravelLogs;

LaravelLogs::hello();
```

## Testing

```bash
composer test
```
